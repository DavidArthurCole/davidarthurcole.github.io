# Site Redesign - Astro Rebuild Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

> **Note:** Per CLAUDE.md, git commits are managed by the user. Omit all `git add` / `git commit` steps.

**Goal:** Replace the existing Vue 3 + Vite portfolio with a ground-up Astro 5 rebuild positioned as a ServiceNow developer/architect site for David Cole.

**Architecture:** Astro 5 generates a fully static site output to `docs/` for GitHub Pages. Vue 3 is used only for islands that need client-side interactivity (sticky nav with scroll spy, skill card hover overlay). All content lives in a single typed data file. The ServiceNow section is the flagship and gets the most visual real estate.

**Tech Stack:** Astro 5, Vue 3, Tailwind CSS v3 (`@astrojs/tailwind`), TypeScript strict, Node.js (compaction script only)

## Task 4: sys_update_xml Compaction Script

**Files:**
- Create: `scripts/compact-sn-export.js`

This script is run once before or during development to populate `src/data/sn-export-compact.json` with filtered ServiceNow records.

- [ ] **Step 1: Write `scripts/compact-sn-export.js`**

```js
// @ts-check
const fs = require('fs');
const path = require('path');

const INPUT = path.join(__dirname, '..', 'sys_update_xml', 'sys_update_xml.json');
const OUTPUT = path.join(__dirname, '..', 'src', 'data', 'sn-export-compact.json');

const KEEP_FIELDS = ['name', 'sys_class_name', 'scope', 'description', 'category'];

// Adjust this username if the filter returns 0 results
const USER_HINT = 'david';

if (!fs.existsSync(INPUT)) {
  console.error(`Input not found: ${INPUT}`);
  process.exit(1);
}

console.log('Reading export (this may take a moment for 71 MB)...');
const raw = JSON.parse(fs.readFileSync(INPUT, 'utf-8'));

// Handle array at root, or { records: [...] }, or { result: [...] }
const records = Array.isArray(raw)
  ? raw
  : Array.isArray(raw.records)
  ? raw.records
  : Array.isArray(raw.result)
  ? raw.result
  : [];

if (records.length === 0) {
  console.error(
    'Could not find a records array in the JSON. ' +
    'Top-level keys: ' + Object.keys(raw).join(', ')
  );
  process.exit(1);
}

console.log(`Total records: ${records.length}`);

const compacted = records
  .filter((r) => {
    const created = String(r.sys_created_by || '').toLowerCase();
    const updated = String(r.sys_updated_by || '').toLowerCase();
    return created.includes(USER_HINT) || updated.includes(USER_HINT);
  })
  .map((r) => {
    /** @type {Record<string, string>} */
    const out = {};
    for (const field of KEEP_FIELDS) {
      const val = r[field];
      if (val !== undefined && val !== null && val !== '') {
        out[field] = String(val);
      }
    }
    return out;
  })
  .filter((r) => r.name);

console.log(`Matched ${compacted.length} records for user hint "${USER_HINT}"`);
fs.writeFileSync(OUTPUT, JSON.stringify(compacted, null, 2), 'utf-8');
console.log(`Written to ${OUTPUT}`);
```
